# ブランド

## 名前

**Markuplint**のスペルは、すべての文字が小文字であるか、大文字の**M**以降に小文字が続きます。**L**は大文字になりません。また、**Markup**と**Lint**をスペースなどでわけません。コマンドなどでは前者を使用し、通常は後者を使用することを推奨しています。

:::note

`v2.x`まではイニシャルも含めてすべて小文字でしたが、`v3.0`からイニシャルが大文字に変更されました。

:::

### 略称

略称は**ML**です。接頭辞として使う場合は`ml-`を使用します。
